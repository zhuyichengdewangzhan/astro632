# 我的Astro 安装部署笔记

## 准备工作
1.安装Node.js
  1.通过官网下载Node.js最新版
  2.在终端中运行 `npm i -g nrm`.然后运行`run test`,之后查看速度最快的镜像源（数值越小越快），再使用`nrm use xxx`（我的是taobao最快）确认使用镜像源，以便加快后续下载安装node包的速度
2.通过老师给的安装包安装VS code
3.安装终端（忘记终端是怎么安装了）

在电脑桌面打开终端，运行`npm creat astro@latest`
